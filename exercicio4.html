Reproduza o layout da imagem, organizando o container com duas seções na vertical (menu e apresentação) e duas na horizontal (a seção de apresentação deve ser dividida entre textos e imagem). Reproduza conforme a imagem, organizando e distribuindo os elementos com Flexbox. Não use CSS no arquivo HTML. O Container principal deve ocupar 100% da largura do Viewport. Use Media Query para ajustar o conteúdo do exercício anterior, para visualizações a partir de 992px. A borda é meramente ilustrativa.
